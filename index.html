How to Install

Since this is an Indie app not yet on the App Store, macOS requires a specific first-launch procedure.

1

Download & Unzip

Download the archive and double-click to extract CursorFixerS.app.

Download .zip
2

Open via Right-Click Important

Do not double-click! Instead, Right-Click (or Control+Click) the app icon and select Open from the menu.

macOS will ask: "macOS cannot verify the developer..."
Click Open in the dialog box.
3

Enable Extension

Open Safari settings:
Safari > Settings > Extensions.

Check the box next to CursorFixerS.

Extension not appearing in Safari?

If you are running a developer version of Safari, you might need to allow unsigned extensions.

  • Open Safari Menu > Settings > Advanced.
  • Check "Show Develop menu in menu bar".
  • In the new "Develop" menu, click "Allow Unsigned Extensions".
