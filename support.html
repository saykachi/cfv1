1. Installation & Activation

CRITICAL: How to bypass the "Unidentified Developer" message?

This is the most common issue. The message appears because the application is not from the App Store. Follow these instructions carefully:

Step-by-Step Guide (FIRST LAUNCH ONLY):

  1. Locate the CursorFixerS.app file in Finder.
  2. DO NOT DOUBLE-CLICK! **Right-click** (or Control + Click) the icon.
  3. In the menu that appears, select **Open**.
  4. In the following warning window, click **Open**.

This is a one-time process. macOS will remember the application afterwards.

Extension does not appear in Safari Settings

Possible causes and solutions:

  • First Launch: Ensure you have launched the main **CursorFixerS.app** at least once.
  • Restart Safari: Fully quit Safari (**Cmd + Q**) and relaunch.
  • Check macOS: The extension requires macOS 12 (Monterey) or newer. Ensure your system is updated.
  • Develop Menu: If you are using beta versions of Safari, go to the **Develop** menu and ensure **"Allow Unsigned Extensions"** is enabled.
Does the app need to stay in the Applications folder?

Yes, the main application file (CursorFixerS.app) must remain on your system, ideally in the **Applications** folder, to keep the Safari Extension active. If you delete the app, the extension will stop functioning until you reinstall it.

How does PiP Force ensure Picture-in-Picture works everywhere?

Some video providers try to disable the native PiP button in Safari. Our extension circumvents this by directly invoking the browser's Picture-in-Picture API (`requestPictureInPicture()`) on the active HTML5 video element.

How is the "Clean Screenshot" function different?

Our Clean Screenshot feature temporarily applies quick CSS styles to hide all overlaid UI elements (play buttons, time bars, social share icons, etc.) before the operating system captures the screen.

3. Troubleshooting & Known Issues

Video playback is lagging after installing the extension

This is extremely rare, but may indicate a conflict.

  • Isolation Test: Temporarily disable CursorFixerS and check if the lag disappears.
  • Conflict Check: Check for conflicts with other video-managing extensions.
  • Hardware Acceleration: Ensure Safari's hardware acceleration is enabled.
Hotkeys are not working on certain sites

Ensure the video player is active: click your mouse inside the video frame.
