We've completely rewritten the ad-skipping engine. It's no longer just a clicker, but an intelligent protection system designed to keep your viewing experience seamless.

AdBlock Revolution
  • YouTube Terminator: Implemented `MutationObserver` algorithm. YouTube ads are destroyed instantly (when `ad-showing` class appears) before you even notice them.
  • Universal Smart Skip: Intelligent skip button for 3rd party sites (anime, movies). Appears only when truly needed.
  • Auto-Killer Mode: Added "Auto Skip" option. If 100% confident, the extension skips ads automatically after 0.8s.
  • Deep Scan Tech: Scans surrounding elements for marker words ("Ad", "Sponsor") to find ads in non-standard players.
Bug Fixes
  • Movie Rewind Fix: Fixed auto-skip triggering on movies/shorts. Added "Fuse": auto-skip disabled if video > 5 mins.
  • Speed Stick Fix: Fixed speed jumping to 5x or 0.25x when holding keys. Switching is now strictly stepwise.
  • Ghost Button: Fixed Skip button blocking UI when no ad is present. It's now completely invisible when idle.
  • Overlay Conflict: Resolved z-index issues where our button overlapped YouTube controls and subtitles.
Performance & UI
  • Zero-Lag Architecture: `content.js` mouse listener rewritten with `passive: true`. CPU usage dropped to near zero.
  • Shadow DOM Support: Now finds video elements inside deep shadow roots (modern SPA sites).
